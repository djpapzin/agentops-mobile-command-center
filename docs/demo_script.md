# 2-Minute Demo Script

## 0:00 - 0:20 — Problem

"I'm a founder on mobile, and I still need to supervise AI workflows, review PRs, and approve decisions without sitting at a laptop."

## 0:20 - 0:45 — Product

"AgentOps Mobile Command Center turns Telegram into an operations console. I can set a goal, check progress, review pull requests, inspect VM storage, and approve the next action from my phone."

## 0:45 - 1:20 — Live demo flow

1. Send `/goal Ship the hackathon MVP`
2. Send `/status` to show the active goal and recent runs
3. Send `/storage` to show VM health
4. Send `/review_pr https://github.com/example/repo/pull/42`
5. Send `/approve` to approve the generated decision card

## 1:20 - 1:45 — Why it matters

"The router chooses the cheapest capable model first and only escalates to Fireworks or AMD-hosted models when task complexity or accuracy requires it. That saves tokens, reduces cost, and keeps decision quality visible."

## 1:45 - 2:00 — Close

"This is a mobile-first control room for operators who want autonomy without losing oversight."
