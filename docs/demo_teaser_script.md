# Demo teaser script

## Goal
A 12–15 second, mobile-first teaser that shows the product as a real live dashboard, not a slide deck.

## Voiceover
"AgentOps Mobile Command Center is a Telegram-first ops console. The live dashboard shows recent runs, approvals, and the smoke-test button. From mobile, I can review work and approve the next step fast."

## Shot list
1. Live dashboard top section — show the live demo button and recent runs.
2. Live dashboard detail — show approvals and status cards.
3. Mobile dashboard view — show the phone-friendly layout.

## Tone
- concise
- direct
- judge-friendly
- product-real
