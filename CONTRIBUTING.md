# Contributing

Thanks for helping improve AgentOps Mobile Command Center.

## Workflow

1. Create a branch.
2. Make the smallest useful change.
3. Run tests locally.
4. Verify the Docker build still works.
5. Open a PR with a short summary and test notes.

## Quality bar

- keep the app lightweight
- prefer demo fixtures over live credentials
- avoid touching production services unless explicitly approved
- update docs when behavior changes
- include tests for routing and command handling
